# Folloze Sales Doc

Use this skill when a teammate needs a branded Folloze sales or customer lifecycle document and the output should follow the shared design system.

## Status

Starter scaffold only. Add the approved templates, design references, and document-generation workflow before broad team use.

## Suggested Subfolders

- `references/` for templates, sample outputs, and design rules
- `scripts/` for document assembly or export helpers
- `agents/` for skill-specific prompts if the workflow needs them

## Common Output Types

- discovery prep
- stakeholder map
- onboarding plan
- QBR
- renewal prep
- account summary
